# REF - 外部参考资料模板

## 类型一：纯外部链接

```markdown
# REF-{编号}-{资料名称}

| 属性       | 值                             |
| ---------- | ------------------------------ |
| 文档编号   | REF-{编号}                     |
| 类别       | REF - 外部参考资料             |
| 状态       | 有效/废弃                    |
| 录入人     | {当前用户.作者}                         |
| 录入日期   | YYYY-MM-DD                     |

> **编码形式**：默认使用简洁编码（`{类型码}-{三位序号}`，与正文占位符一致）。
> **项目前缀**：仅在当前作用域 `README.md` 已声明 `project_code` 且请求发起方明确确认启用时，在各类编码前加 `{项目编码}-` 前缀（细则见 `references/coding-system.md`）。

## 来源信息

- **类型**：标准/论文/书籍/技术文档/竞品资料/网页/其他
- **来源**：{出处，如组织名称、网站、出版社}
- **链接**：{URL，可以是多个链接}
- **获取日期**：YYYY-MM-DD
- **版本/版次**：{如适用}

## 摘要

用一到两段话概括该资料的核心内容和关键结论。

```

## 使用说明

1. **类型一（外部链接）**：只保留来源信息和摘要
2. **类型二（本地文档）**：头部 + 来源信息 + 原文档内容
3. 状态值从 `有效/废弃` 中选择
4. **编码形式与项目前缀**：见类型一/类型二文首说明
5. **分配编码**：新增 `REF` 前检索当前作用域 `README.md` 全局索引，分配后同步更新（无独立「细项清单」小节时亦同）
