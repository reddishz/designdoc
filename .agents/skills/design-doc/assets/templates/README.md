# 设计文档模板库

本目录仅包含**可复制的文档模板**（填空骨架）。流程图规范、AI 编码约束等见 [辅助指南](../guides/README.md)。

## 模板使用指南

所有模板遵循 SSoT 原则：

- **不得包含代码和具体实现**，只允许规则、约束、逻辑、流程、概念、假定等设计性内容
- 所有明确细项必须分配全局唯一编码（`{类型码}-{三位序号}`）
- 每份文档末尾维护细项编码清单
- 交叉引用使用全局唯一编码，下层可引用上层，上层无需引用下层
- 模板中 `{当前用户.作者}` 表示使用 SKILL.md「当前用户」章节动态注入的作者名

## 使用说明

1. 根据文档层级选择对应模板
2. 替换模板中的占位符（如 `{编号}`、`{文档名称}`）
3. 确保状态值符合标准：`草稿/正式/废弃`
4. 维护细项编码清单和变更记录
5. 遵循 SSoT 原则进行编码和引用
