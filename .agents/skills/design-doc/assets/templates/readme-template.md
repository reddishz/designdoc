## 编码分配工具

### 快速分配编码
1. 查看编码计数器表中的"下一个编号"
2. 使用对应的完整编码（如 `{项目编码}-FR-015`）
3. 更新计数器：将"下一个编号"递增
4. 在文档中添加编码引用

### 检查编码冲突
1. 在 `ued/` 目录中搜索目标编码
2. 确认编码未被使用后再分配
3. 检查文件名是否与编码冲突

## 使用说明

1. **新增文档**：从对应层级的模板创建，更新文档索引表
2. **分配编码**：使用编码计数器快速获取下一个可用编码
3. **更新索引**：创建新细项时，同步更新全局编码索引和文档内细项编码清单
4. **维护计数器**：每次分配编码后，及时更新计数器
