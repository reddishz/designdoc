## 使用说明

1. 替换 `{编号}`、`{文档名称}` 等占位符
2. 状态值从 `草稿/正式/废弃` 中选择
3. 确保所有细项编码全局唯一
4. 关联文档引用对应的 L2 和 L5 文档编号
5. 需求追溯状态从 `待验证/已验证/失败` 中选择
