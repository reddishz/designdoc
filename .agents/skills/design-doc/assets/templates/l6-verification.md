## 使用说明

1. 替换 `{编号}`、`{文档名称}` 等占位符
2. 状态值从 `草稿/正式/废弃` 中选择
3. **编码形式与项目前缀**：见文首说明
4. **本文档细项编码清单**：仅本文档范围；分配前检索 `ued/README.md` 全局索引，分配后同步更新
5. 关联文档引用对应的 L2 和 L5 文档编号
6. 需求追溯状态从 `待验证/已验证/失败` 中选择
