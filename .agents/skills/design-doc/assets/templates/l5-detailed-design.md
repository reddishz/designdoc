## 使用说明

1. 替换 `{编号}`、`{文档名称}` 等占位符
2. 状态值从 `草稿/正式/废弃` 中选择
3. **编码形式与项目前缀**：见文首说明
4. **本文档细项编码清单**：仅本文档范围；分配前检索当前作用域 `README.md` 全局索引，分配后同步更新
5. 关联文档引用对应的 L4 文档编号
6. 不写类成员、函数签名、配置片段等实现内容，只写算法、流程、逻辑和约束
